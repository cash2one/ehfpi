EHF-pathogen Network
A network representation of the EHF genes and pathogens is provided based on the custom gene list. You can provide gene symbols in three ways:
  • EHFPI guidance tree
  • Input a gene list
    • Genes should be separated by comma, e.g., cab5,cbx4, or one gene per line.
  • Upload a text file
    • text file, gene separated by comma e.g., cab5,cbx4. Or one gene per line.
Please note that it may take minutes to load the network if you submit too large amount of genes. In the EHF-pathogen network, dots represent genes and stars represent pathogens. An edge connecting a gene and a pathogen represents this gene is an EHF of this pathogen. The edges are differentially colored according the phenotypes. Below is an example figure of EHF-Brucella abortus(Brucella melitensis) netwrok.
The left top window is a result display, showing all the pathogen(s) that share(s) the designated gene(s) as EHF(s) and corresponding numbers.
The following operations are supported in the network window:
  • Filter the result.
    • You can select one or several pathogen species in the result display window. The corresponding nodes (stars and dots) will be highlighted.
  • Mouse over a node's label to select a node and its connections.
    • The edges connected to the selected node will be highlighted. If you point at a gene (dot), the label will display its full name, and its related pathogen information will be shown in the left bottom panel. If you point at a pathogen(star), the label will display strains/subtypes of the pathogen, and the characteristics of its EHF genes (common/specific) will be summarized and shown in the left bottom panel.
  • click on the 'x' link to delete a node.
    • You can hide the pathogen node by clicking the 'x' link, if the gene node connected to this node is specific (means only connected to one pathogen node), the node will hide too.
  • Drag nodes around.
    • You can drag the node to the place you want in the window for better view.
  • Zoom and pan the figure.
    • You can zoom the figure with mouse wheel, or use the 'zoom in' and 'zoom out' button on the right top of network window. You can also pan the graph.
  • Search the network.
    • You can search gene symbol(s) or pathogen(s) in the network. When an exact match is found, the corresponding nodes will be highlighted.
  • Save the network as an image or csv file.
    • Only visible filed of the network window is saved, but csv file contains all information.
Please note that single nodes may not be placed in the visible scope of the network window. You can zoom in to see all nodes.
EHF overlap analysis
A EHF overlap analysis displays the common EHF genes of pathogens in a heatmap representation. You can run the analysis in two aspects:
  • Taxonomy
    • Common EHF genes of pathogen(s) will be custom displayed at four different taxonomy levels (i.e. kingdom, family, specie, and strain). Please note that for viral pathogens, the analysis at genus level is not provided, and for fungal and bacteria l pathogens, the analysis at family level is not provided.
  • Article
    • Common EHF genes for a specific will be custom displayed based on the published articles that reports the RNAi screen results pathogen. To distinguish different pathogen species tested in one article, we further add pathogen information to the guidance tree.

Below is an example of EHF overlap analysis result using taxonomy. In the heatmap, squares are differentially colored according to the number of shared EHF genes for a pair of pathogens (names shall be notified with mouse arrow placed on the square). And the exact number is placed in the center of each square, with the detailed information displayed in the left bottom panel as in a 'Common Genes' list. This window can be moved in the page at your convenience.
By default, the result is generated and ordered by name at kingdom level. You can also change the dominant display order. Now, EHFPI supports three orders:
  • name
  • common gene number
  • pathogen group
    • e.g., species of the same family.
Clicking the square to see the corresponding result page. See search for more help. Please note that the size of guidance tree window is adjustable and supports search for further information.

What's more, you can view the list of EHF genes and their related pathogen lists. The table below list the EHF genes and their interacted pathogen number and pathogen list.
You can download specific or all entries for further analysis. You can also filter the result, the search field supports searching multiple gene symbols separated by comma or semicolon. Leave the field empty to return all results.

We also provide the distribution statistics of EHFs. Besides the statistics of all EHF genes, statistics of primary hits is also given. In the column graph, x axis represents interacted pathogen number, and y axis represents EHF number. For example, in the first column graph, there are 716 EHF genes who interact with 2 pathogens (note the pathogens for different EHFs may be different). This graph provides an overview of overlap statistics. You can save the column graph as an image or a pdf file.

The column graph doesn't provide detailed gene information, so we present the gene list in a separate table. Generally EHFs with more interacted pathogens get more attention.
GEA Analysis
EHFPI provides direct links to DAVID to perform gene enrichment analysis. You can choose the EHF genes of a specific pathogen or from a screen reported by a specific publication to run the analysis. In the filter window, the number mark at the end of each guidance item represents the total number of EHFPI accessions under this statistical category. Please note that the size of filter window is adjustable.
We provide GO and KEGG pathway analysis by default. For more details, please refer to DAVID help. Please note that DAVID API has limitations on the total number of genes for submission!

To provide the user more alternatives, we also link to other external tools such as GOEAST, PANTHER, g:Profiler, Reactome, Pathway Interaction Database and Pathway Commons. You can copy the gene symbols and go to a specific link for analysis.
PIPs Analysis
PIPs analysis provides information on whether or not the protein product of an EHF gene is experimentally validated as a pathogen interacting protein. The PIPs data are from VirHostNet and VirusMINT. You can provide a gene list or choose EHF genes of specific pathogen(s) to view the analysis result.
The result page includes a summary of the analysis and a list of brief information regarding the EHFs whose protein products are also PIPs.
You can also download PIPs analysis data or perform GEA, network analysis and view PPI network based on the result.
Below is an example PPI network, the protein-protein interaction data are from HPRD.
The following operations are supported in the ppi network window:
  • Click a node to select a node and its connections.
    • When clicking a node, the node and its neighbourhood (edges and nodes connected to the node) are highlighted. Moreover, a tip which shows the detailed protein and related nodes information will be displayed.
  • Click an edge to select an edge.
    • When clicking an edge, the experiment type and literature information will be displayed in a popover tip window.
  • Drag nodes around.
    • You can drag a single node or a group of nodes (select a rectangle area with left mouse button) to the place you want in the window for better view.
  • Zoom and pan the figure.
    • You can zoom and pan the figure with mouse wheel, or with the buttons on the left top of the network window.
  • Search the network.
    • You can search gene symbol, HPRD id or Refseq id in the network. When an exact match is found, the corresponding node and its neighborhood will be highlighted.
  • Save the network as an image.
    • You can save graph in current viewport or entire graph as an png image.
  • Navigator the graph.
    • You can navigate the graph with the widget on the right bottom of the window. This widget shows a bird's eye view of the graph such that the entire graph is visible. Using this widget, the user can zoom and pan the graph.
Drug Target Analysis
Drug target analysis provides specialized information for pathogens whose EHF genes or protein products are also validated targets of FDA-approved drugs and other drugs(experimental, investigational, withdrawn, nutraceutical). The EHF-drug connection is defined as the EHF gene of a specific pathogen species and the targets of a specific drug or small bioactive chemical compound/molecule share at least one gene/protein product.
The drug target data are from DrugBank (only drug with known pharmacological action is included). You can choose EHF genes of specific pathogen(s) to view the analysis result.
The result page includes a summary of the analysis and a list of brief information of pathogen species, drug target and drug.
Here is an example of result for Brucella abortus(Brucella melitensis). You may find that besides submitted pathogen specie(s), the EHF-drug connections for additional pathogen specie(s) who share EHF genes of Brucella abortus(Brucella melitensis) are also presented.
You can see the detailed drug information by clicking the DrugBank ID external link.
To view result list of specific pathogen species, you can filter the table by clicking the 'Filter' button on the right top of the table. Submitted pathogens are selected by default.
You can also download the results of interest. Or you can view the EHF-drug connections in a network representation. The EHF-drug network is built in this way: an edge is connected between a specific pathogen and a drug if the EHF gene of this pathogen specie is a target of the drug.
Here is an example of EHF-drug network for Brucella abortus(Brucella melitensis). You may find that besides submitted pathogen specie(s), the EHF-drug connections for additional pathogen specie(s) who share EHF genes of Brucella abortus(Brucella melitensis) are also presented.
The left top window is a result display, showing all the pathogen(s) that share(s) the designated drug(s) and corresponding numbers.
The following operations are supported in the network window:
  • Filter the result.
    • You can select one or several pathogen species in the result display window. The corresponding nodes (triangles and dots) will be highlighted. And the related drug list will displayed in the left bottom panel.
  • Mouse over a node's label to select a node and its connections.
    • The edges connected to the selected node will be highlighted. If you point at a drug (dot), the tip will display its DrugBank ID, and its related pathogen information will be shown in the left bottom panel, as well as the drug target that links the drug and the pathogen. If you point at a pathogen(triangle), the tip will display strains/subtypes of the pathogen, and the related drug information (approved and others) will be summarized and shown in the left bottom panel.
  • click on the 'x' link to delete a node.
    • You can hide the pathogen node by clicking the 'x' link, if the drug node connected to this node is specific (means only connected to one pathogen node), the node will hide too.
  • Drag nodes around.
    • You can drag the node to the place you want in the window for better view.
  • Zoom and pan the figure.
    • You can zoom the figure with mouse wheel, or use the 'zoom in' and 'zoom out' button on the right top of network window. You can also pan the graph.
  • Search the network.
    • You can search drug(name and DrugBank ID) or pathogen(s) in the network. When an exact match is found, the corresponding nodes will be highlighted.
  • Save the network as an image.
    • Only visible filed of the network window is saved.
  • Click edge to get drug target info.
    • Since there exists a hidden layer(drug target) in the network, you can also get drug target information by clicking the edge between pathogen node and drug node.
Please note that single nodes may not be placed in the visible scope of the network window. You can zoom in to see all nodes.
GWAS Candidate Gene Analysis
The NHGRI GWAS Catalog supports Genome-wide association study (GWAS) that screens for the candidate genes, of which the common genetic variants, e.g., single-nucleotide polymorphisms (SNPs), in different individuals are associated with a trait/disease. EHFPI provides further annotation/interpretation of EHF genes as candidate disease/traits genes identified from Genome Wide Association Studies. You can choose the EHF genes of specific pathogen(s) to perform the analysis.
The result page includes a summary of the analysis and a list of brief information.
You can also download GWAS data. A detailed description of each field can be found here.
