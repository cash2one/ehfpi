Currently EHFPI provide six analysis functions.
EHF-pathogen Network
  • Overlay EHFPI genes shared by different pathogens at designated level, and displayed onto a pathogen-EHF network.
EHF Overlap Analysis
  • Overlay EHFPI genes shared by screens/publications reported from different labs, as displayed in a Heatmap representation of screen/publication titles(taxonomy or article).
Gene Enrichment Analysis
  • DAVID for function annotation of EHFPI genes, as in GO Bioprocess and KEGG pathway.
  • GOEAST, PANTHER and g:Profiler for function annotation of EHFPI genes, as in GO Bioprocess.
  • Reactome, Pathway Interaction Database and Pathway Commons for function annotation of EHFPI genes, as in pathway.
Pathogen Interacting Proteins (PIPs) Analysis
  • List of EHFPI genes whose protein products are experimentally validated as pathogen interacting proteins.
  • Verification of a custom-provided set of gene as EHFs and/or PIPs, with detailed information concerning related pathogen type(s) and screen(s).
(Note: Reference PIP data for designated pathogens are from VirHostNet and VirusMINT)
Drug Target Analysis
  • Network representation of EHFPI genes that are experimentally validated as drug targets.
(Note:Reference drug data are from DrugBank)
GWAS Candidate Gene Analysis
  • Identification and statistics of EHFPI genes as Genome-wide association study (GWAS) candidate genes, of which the common genetic variants, e.g., single-nucleotide polymorphisms (SNPs), in different individuals are associated with a trait/disease.
(Note:Reference GWAS data are from gwastudies)
